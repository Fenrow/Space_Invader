# Space_Invader
Space Invader Game

Sterowanie:
Spacja - strzał
Strzałka w lewo - ruch statku w lewo
Strzałka w prawo - ruch statku w prawo
Q - wyjście
G - rozpoczęcie nowej gry

Control:
Spacebar - shot
Left arrow - ship movement to the left
Right arrow - move the ship to the right
Q - exit
G - start a new game
